Linux 0.00 
================================================================================


下载
---------------------------------------------------------------------------

从网站上下载源代码，

.. prompt:: bash $

    git clone https://gitee.com/guojunos/Linux000.git


- ``Linux000 code/`` 目录下是源码，经修改后可编译成功。
- ``windows`` 下可直接运行 ``Linux000.bat``
- ``linux000_gui.bxrc`` 是 ``Bochs`` 配置文件
- ``Image`` 是生成的 ``Linux 0.00`` 的二进制文件，被加载到软驱，由 ``Bochs`` 从配置文件读取并启动执行。


编译
---------------------------------------------------------------------------


参见 ``Linux000 code/`` 目录下的 ``Makefile``，执行

.. prompt:: bash $

    make all



调试
---------------------------------------------------------------------------

``windows`` 下可运行 ``Linux000.bat`` ，会直接弹出调试界面。
