5. 实验报告¶

这里的实验报告建议用 Markdown 进行书写。

5.1. Linux 实验报告¶

总共有 8 次实验报告，其格式如下所示：

|──001-xxx
    ├─001_lab01
    │  │  reports.md
    │  │
    │  └─images
    ├─001_lab02
    │  │  reports.md
    │  │
    │  └─images
    ├─001_lab03
    │  │  reports.md
    │  │
    │  ├─code
    │  └─images
    ├─001_lab04
    │  │  reports.md
    │  │
    │  ├─code
    │  └─images
    ├─001_lab05
    │  │  reports.md
    │  │
    │  ├─code
    │  └─images
    ├─001_lab06
    │  │  reports.md
    │  │
    │  ├─code
    │  └─images
    ├─001_lab07
    │  │  reports.md
    │  │
    │  ├─code
    │  └─images
    └─001_lab08
        │  reports.md
        │
        ├─code
        └─images

  • 001 是 id 号，xxx 是姓名，详见名单

  • reports.md 是 Markdown 的实验报告

  • images 子目录存放的是报告里用到的图片

  • code 从实验 3 开始提交的存放代码的目录

例如： 001 号同学第一次实验提交的报告为 001-xxx-lab01.rar ，这个压缩文件里应是如下形式：

|──001-xxx-lab01.rar
    ├─001_lab01
    │  │  reports.md
    │  │
    │  └─images
    │          img1.png
    │          img2.png

警告

引用图片时，不要使用绝对路径，要使用相对路径。

例如：

![alt text](.\images\img1.png)

5.2. 读书笔记报告¶

读书笔记如果是电子版的 Markdown 格式报告，格式与实验一的报告格式相同， 把 lab01 改为 read01 即可，如下：

|──001-xxx-read01.rar
    ├─001_read01
    │  │  reports.md
    │  │
    │  └─images
    │          img1.png
    │          img2.png
